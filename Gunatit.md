# Image Requirements: Divine Lineage Section
**Page:** About YDS Canada (`/about/yds-canada`)
**Section:** Akshar Purushottam Upasana - Divine Lineage Timeline

---

## Overview
This document outlines the image requirements for the 6 divine figures in the spiritual lineage. Each image will be displayed in a beautiful modal when users click on the respective saint's card.

---

## Image Specifications

### General Requirements for All Images:
- **Aspect Ratio:** Portrait orientation (3:4 or similar)
- **Recommended Size:** 800px Ã— 1000px (minimum)
- **Format:** JPG or PNG
- **Quality:** High resolution for clear viewing
- **Background:** Can have background, will be displayed in a modern gradient container
- **Style:** Respectful, dignified, traditional spiritual imagery

### Display Context:
- Images will be shown in a **264px to 320px height hero section** at the top of each modal
- The image container has a **beautiful gradient background** (gold for #1, purple for others)
- **Fallback:** If image is not available, a stylish numbered badge will be shown with "Image Coming Soon" text

1. Prepare your 6 portrait images according to specifications above
2. Rename them exactly as specified:
   - `placeholder-saint-1.jpg` (Bhagwan Swaminarayan)
   - `placeholder-saint-2.jpg` (Gunatitanand Swamiji)
   - `placeholder-saint-3.jpg` (Shastriji Maharaj)

3. Place all images in `/public/` folder
4. Refresh the website - images will automatically appear in the modals!

## Fallback Behavior

If an image is not found:
- A beautiful **numbered circular badge** will be displayed
- The badge uses the same gradient as the card (gold for #1, purple for others)
- Text "Image Coming Soon" will appear below
- This ensures the page still looks polished even without images

## Tips for Best Results

1. **High Quality:** Use the highest quality images available
2. **Proper Cropping:** Ensure the subject is centered and prominent
3. **Consistent Style:** Try to maintain visual consistency across all 6 images
4. **Respectful Content:** These are revered spiritual figures - ensure dignity in imagery
5. **Test Display:** After adding images, test on both desktop and mobile
